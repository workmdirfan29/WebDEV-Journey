Welcome to HTML Basics

Here’s a simple explanation of what each part of the code does:

1. <!DOCTYPE html>

This tells the browser that we are using HTML5. It must be the very first line of our document, so the browser knows how to read it correctly.

2. <html lang="en">

The <html> tag is the start of our webpage. The lang="en" part tells the browser that the page is in English, which helps with accessibility and search engines.

3. <head>

The <head> section contains important information about the webpage, but it doesn’t appear directly on the page you see in the browser.

Inside the <head>:

  • <meta charset="UTF-8">: This line tells the browser to use UTF-8 encoding, which helps display characters from many languages correctly.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: This makes sure our webpage looks good on mobile devices by setting the correct zoom and width.
  • <title>Document</title>: This sets the name of the webpage that shows up in the browser tab (it’s not shown on the webpage itself, just in the tab).

4. <body>

The <body> section holds everything that will appear on the webpage, like text, images, and links. This is where you add the visible content.

5. Comments

Comments are notes you can add inside your code to explain things. These don’t show up on the webpage. They look like this:

<!-- This is a comment -->

For example, you can use comments to remind yourself what a piece of code does or to explain things to others who read your code.
